Project Motivation

Why we built this

In Indian kitchens — and kitchens worldwide — imprecise measurements lead to inconsistent results. Traditional "andaaz" (estimation) cooking, while beautiful in tradition, fails when precision matters: baking, nutrition tracking, or scaling recipes. We set out to solve this with affordable, accessible IoT technology.

01 — PROBLEM
Kitchen Precision Gap
Most home cooks rely on volume measurements or estimation. Even a 10% error in flour can ruin a cake. Precision tools exist, but they're passive — they don't guide you.
02 — SOLUTION
IoT-Guided Ingredient Flow
Our system combines a strain-gauge load cell, HX711 amplifier, Arduino, Firebase, and a web UI to create an interactive cooking guide that reacts to your scale in real time.
03 — IMPACT
Precision for Everyone
Built at under ₹800 total, this system brings lab-grade measurement to any kitchen. Cloud logging enables nutrition tracking, recipe calibration, and dietary compliance over time.

Technical Report

Project Documentation

📋

Abstract

This project presents a 1 kg IoT-enabled kitchen scale developed for ENR206 — Sensors and Instruments. Using a strain-gauge load cell, HX711 24-bit ADC, Arduino Uno, I2C LCD, Firebase RTDB, and a web dashboard, the system provides real-time ingredient weighing with cloud logging and an interactive recipe-guided UI. The scale achieves ±1g resolution with a calibration factor of 2165.0.

🔬

Sensor Principle

The load cell operates on the principle of the Wheatstone bridge. Applied load causes micro-deformation in strain gauges bonded to the flexural element, changing their resistance. The HX711 amplifies this differential voltage (typically 0–20 mV full scale) and digitizes it at 24-bit resolution, achieving 0.1g precision before calibration. Calibration factor 2165.0 was empirically determined using a known 172g reference mass.

⚙

Hardware Components

  • Arduino Uno R3 (ATmega328P, 16 MHz)
  • HX711 24-bit ADC Load Cell Amplifier
  • 1 kg Strain Gauge Load Cell (4-wire)
  • 16×2 LCD with I2C backpack (0x27)
  • Jumper wires, breadboard, MDF base
  • USB cable for serial communication
💻

Software Stack

  • Arduino IDE — HX711_ADC, LiquidCrystal_I2C libraries
  • Python 3 — pyserial, firebase-admin SDK
  • Firebase Realtime Database (RTDB)
  • Vanilla HTML/CSS/JS frontend dashboard
  • Firebase JS SDK v9 for real-time listener
  • Serial baud: 57600, print interval: 500ms
📊

Results & Observations

The system demonstrates stable readings within ±1–2g across the 0–1000g range. Tare function successfully zeroes the display for consecutive measurements. LCD updates every 500ms with 1 decimal precision. Firebase latency measured at <300ms on local network. The calibration value of 2165.0 was validated against multiple reference objects including a known 172g smartphone.

Arduino Firmware

HX711 load cell with I2C LCD — calibration factor 2165.0

scale_lcd.ino 
/*  HX711 Load Cell — With 16x2 I2C LCD
    ENR206 · Group: Parthiv, Aadi, Chintan, Aarav  */

#include <HX711_ADC.h>
#include <LiquidCrystal_I2C.h>
#if defined(ESP8266) || defined(ESP32) || defined(AVR)
#include <EEPROM.h>
#endif

// ── Pins ──────────────────────────────────────
const int HX711_dout = 4;
const int HX711_sck  = 5;

// ── Objects ───────────────────────────────────
HX711_ADC LoadCell(HX711_dout, HX711_sck);
LiquidCrystal_I2C lcd(0x27, 16, 2);

float calibrationValue = 2165.0;
unsigned long t = 0;

void setup() {
  Serial.begin(57600);
  delay(10);

  lcd.init();
  lcd.backlight();
  lcd.setCursor(0, 0); lcd.print("Load Cell Ready");
  lcd.setCursor(0, 1); lcd.print("Initializing...");

  LoadCell.begin();
  LoadCell.start(2000, true);

  if (LoadCell.getTareTimeoutFlag()) {
    Serial.println("Timeout, check wiring!");
    lcd.clear(); lcd.print("Timeout!");
    while (1);
  } else {
    LoadCell.setCalFactor(calibrationValue);
    lcd.clear(); lcd.print("Scale Ready!");
    lcd.setCursor(0,1); lcd.print("Send 't' to tare");
    delay(1500);
  }
}

void loop() {
  static boolean newDataReady = 0;
  const int printInterval = 500;

  if (LoadCell.update()) newDataReady = true;

  if (newDataReady && millis() > t + printInterval) {
    float weight = LoadCell.getData();
    Serial.print("Weight (g): ");
    Serial.println(weight);
    lcd.clear();
    lcd.setCursor(0,0); lcd.print("Weight:");
    lcd.setCursor(0,1); lcd.print(weight, 1); lcd.print(" g");
    newDataReady = 0;
    t = millis();
  }

  if (Serial.available() > 0) {
    char inByte = Serial.read();
    if (inByte == 't') {
      LoadCell.tareNoDelay();
      lcd.clear(); lcd.print("Taring...");
    }
  }

  if (LoadCell.getTareStatus() == true) {
    lcd.clear(); lcd.print("Tare Complete!");
    delay(1000);
  }
}
